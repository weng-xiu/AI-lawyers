## 启动计划

### 1. 检查环境依赖
- 确保系统已安装Java 1.8或以上版本
- 确保系统已安装Maven 3.6.0或以上版本
- 确保系统已安装Node.js 14.0或以上版本
- 确保系统已安装npm 6.0或以上版本
- 确保数据库服务（MySQL）已启动并可用

### 2. 启动后端服务
- **模块**：ai-admin
- **启动方式**：使用Maven运行spring-boot:run命令
- **命令**：`mvn spring-boot:run`
- **工作目录**：`d:\project_job\AI-lawyers\ai-admin`
- **预期结果**：后端服务启动成功，控制台显示启动成功信息，端口为8080
- **验证方式**：访问http://localhost:8080，应该看到后端服务的默认页面或API文档

### 3. 启动管理前端
- **模块**：ai-ui
- **启动方式**：使用npm运行dev命令
- **命令**：`npm run dev`
- **工作目录**：`d:\project_job\AI-lawyers\ai-ui`
- **预期结果**：管理前端启动成功，控制台显示访问地址（如http://localhost:80）
- **验证方式**：访问控制台显示的地址，应该看到管理前端的登录页面

### 4. 启动用户前端
- **模块**：aiuser-ui
- **启动方式**：使用npm运行dev命令
- **命令**：`npm run dev`
- **工作目录**：`d:\project_job\AI-lawyers\aiuser-ui`
- **预期结果**：用户前端启动成功，控制台显示访问地址（如http://localhost:81）
- **验证方式**：访问控制台显示的地址，应该看到用户前端的首页

### 5. 注意事项
- 严格按照顺序启动服务，确保后端服务启动成功后再启动前端服务
- 启动过程中注意查看控制台输出，如有错误信息及时处理
- 确保各个服务的端口不冲突
- 前端服务依赖后端API，如后端服务未正常启动，前端服务可能无法正常访问
